GDLite - Geometry Dash Lite

Overview

GDLite is a specialized version of the GDLite bot designed specifically for Geometry Dash Lite. This version offers streamlined features and commands to enhance your gameplay experience.

Features

GDLite for Geometry Dash Lite includes the following features:

  • Level Information: Retrieve detailed information about levels in Geometry Dash Lite.
  • Embed Creation: Create and customize embedded messages related to your gameplay.
  • Command Support: Utilizes both text and slash commands for ease of use.

Available Commands

The following commands are available in GDLite for Geometry Dash Lite:

  • !level [id]: Fetch information about a specific level using the provided ID.
  • /level [id]: An alternative command for retrieving level details using the ID.

Setup Instructions

To set up GDLite for Geometry Dash Lite:

  1. Invite the bot to your server using the appropriate invite link.
  2. Ensure that the bot has the required permissions to function correctly, including sending messages and creating embeds.
  3. Follow any additional setup steps as outlined in the main documentation.
