Installation Guide

To install GDLite, follow these steps:

  1. Invite the bot to your server using the invite link provided in the main documentation.
  2. Grant the bot the necessary permissions to operate effectively, such as sending messages and creating embeds.
  3. Configure any additional settings as needed based on your server's requirements.

Available Commands

GDLite supports several commands to interact with the Geometry Dash Lite game:

  • !level [id]: Retrieves information about a level with the specified ID.
  • /level [id]: An alternative command for retrieving information about a level with the specified ID.

Troubleshooting

If you encounter any issues while using GDLite, consider the following steps to resolve common problems:

  • Ensure the bot has the correct permissions in your server.
  • Verify that the commands are being used correctly with valid parameters.
  • Check the bot's status and logs for any error messages that could indicate issues.
