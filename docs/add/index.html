Adding GDLite to Your Discord Server

Follow these steps to add GDLite to your Discord server:

  1. Click the button below to invite GDLite to your server.
  2. Choose the server you want to add the bot to.
  3. Authorize the bot by granting the necessary permissions.
  4. Complete the CAPTCHA if prompted to ensure security.
Invite GDLite

OAuth2 Permissions

To ensure GDLite functions properly, it requires the following permissions:

  • Send Messages: Allows the bot to send messages in channels.
  • Embed Links: Enables the bot to create embedded messages.
  • Read Message History: Allows the bot to read previous messages in the channel.

Troubleshooting

If you encounter issues while adding the bot, consider the following tips:

  • Ensure you have the "Manage Server" permission in the server you are adding the bot to.
  • Check if the OAuth2 URL is correct and includes the necessary permissions.
  • Contact support if you experience any persistent problems.
